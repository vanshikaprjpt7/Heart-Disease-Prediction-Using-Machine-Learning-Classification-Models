# Heart-Disease-Prediction-Using-Machine-Learning-Classification-Models

This project aims to predict the likelihood of heart disease in patients using various machine learning classification algorithms. It includes data preprocessing, exploratory data analysis (EDA) with visualizations, and model evaluation.

✏️PARAMETERS:
age , gender , chest pain type , BP and Cholesterol levels 

✏️LANGUAGE: 
Python

✏️LIBRARIES: 
pandas & numpy (DATA MANIPULATION)
matplotlib & seaborn (DATA VISUALIZATION) 
scikit-learn (ML MODELING)
